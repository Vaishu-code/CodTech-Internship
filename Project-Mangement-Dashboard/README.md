## Project Description

The Project Management Dashboard is a web application designed to help users organize and manage multiple projects. Users can add, edit, delete, and track project status. The dashboard displays project statistics and stores all data using Local Storage.

---

## Features

- Add New Project
- Edit Project Details
- Delete Project
- Dashboard Statistics
- Project Status Tracking
- Local Storage Support
- Responsive User Interface
