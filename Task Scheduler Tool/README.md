## Project Description

The Task Scheduler Tool is a web application developed using HTML, CSS, and JavaScript. It helps users organize their daily tasks efficiently by allowing them to create, edit, delete, and complete tasks. Task information is stored using the browser's Local Storage so that data remains available even after refreshing the page.

---

## Features

- Add New Task
- Edit Existing Task
- Delete Task
- Mark Task as Completed
- Select Date and Time
- Store Data using Local Storage
- Responsive User Interface
