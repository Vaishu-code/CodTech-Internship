## Project Description

The Digital Journal Platform is a web-based journal application that enables users to record daily thoughts and experiences. Users can create, edit, search, and delete journal entries while selecting their mood and date. All journal data is stored using Local Storage.

---

## Features

- Create Journal Entries
- Edit Journal Entries
- Delete Journal Entries
- Search Journal Entries
- Mood Selection
- Date Selection
- Local Storage Support
